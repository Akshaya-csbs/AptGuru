An AI-powered aptitude chatbot that helps students solve questions step-by-step with clear explanations, shortcuts, and logic. It supports text and image-based questions and explains why each option is right or wrong. The chatbot teaches aptitude topics like Quantitative, Logical, Verbal, and Data Interpretation using fun examples, mini stories, and memory tricks. Learning is made engaging through daily challenges, timed quizzes, hints, levels, and rewards. It creates a stress-free environment where students learn at their own pace and improve faster with instant feedback
